Programação com Python e Java

Nível Básico:


Fundamentos de Programação:


  • Introdução à lógica de programação e algoritmos
  • Conceitos básicos de variáveis, tipos de dados e estruturas de controle (condicionais e loops)
  • Aprender a usar as principais estruturas de dados, como arrays, listas e dicionários

Introdução à Python:


  • Instalação e configuração do ambiente Python
  • Sintaxe básica e características da linguagem
  • Manipulação de strings e input/output básico
  • Funções e modularização
  • Trabalhar com arquivos

Nível Intermediário:


Programação Orientada a Objetos em Python:

  • Conceitos fundamentais de orientação a objetos
  • Classes, objetos, herança e polimorfismo
  • Encapsulamento, abstração e herança
  • Princípios de design orientado a objetos

Desenvolvimento de Aplicações com Python:


  • Utilização de bibliotecas e frameworks populares, como Flask ou Django
  • Manipulação de bancos de dados usando SQL ou ORMs
  • Trabalhar com APIs e integração de serviços
  • Desenvolvimento de projetos práticos para aplicar os conhecimentos adquiridos
  • Manipulação de arquivos JSON e XML
  • Programação concorrente e paralela
  • Tratamento de exceções avançado e assertivas
  • Bibliotecas populares para ciência de dados, como NumPy, Pandas e Matplotlib
  • Criação de interfaces gráficas com Tkinter ou PyQt
  • Introdução ao desenvolvimento de jogos com Pygame

Fundamentos de Java:


  • Instalação e configuração do ambiente Java
  • Sintaxe básica e características da linguagem
  • Conceitos de classes, objetos, herança, polimorfismo e encapsulamento em Java
  • Tratamento de exceções
  • Programação Orientada a Objetos em Java

  • Aprofundamento nos conceitos de orientação a objetos em Java
  • Relacionamentos entre classes, interfaces e pacotes
  • Uso de bibliotecas padrão do Java

Nível Intermediário:


Desenvolvimento de Aplicações com Java:


  • Desenvolvimento de aplicações desktop e web utilizando frameworks JavaFX, Spring ou Java EE
  • Manipulação de bancos de dados com JDBC ou ORMs
  • Trabalhar com APIs externas e serviços web
  • Manipulação de arquivos JSON e XML
  • Programação concorrente e paralela com threads e sincronização
  • Tratamento de exceções avançado e assertivas
  • Uso de coleções avançadas, como Set, Map e Queue
  • Introdução ao desenvolvimento Android com Java e o uso do SDK do Android
  • Desenvolvimento com Java EE, Java Servlets, JavaServer Pages (JSP) e Java Persistence API (JPA)

Nível Avançado:


  • Conceitos avançados de programação, como concorrência, threads e programação assíncrona
  • Padrões de projeto (Design Patterns) e boas práticas de desenvolvimento
  • Aprofundamento em IoT
